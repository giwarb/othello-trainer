# T081 設計依頼(動作確認用デモ)

## 背景
T079 のレビューで、`app/src/analysis/` 配下の「盤面で並べる」タブと「盤面を自由配置」タブの間で、
着手を積み上げて盤面状態を再構築するロジックが重複している可能性が指摘された。

## 依頼内容
`app/src/analysis/` を調査し、両タブの着手積み上げロジックの重複箇所を特定した上で、
それらを共通ヘルパー(関数またはモジュール)に抽出する設計案を提示してください。

以下を含めてください。
- (a) 推奨する設計とその理由
- (b) 検討した代替案と却下理由
- (c) 実装タスクへの分割案(各タスクの変更対象ファイル・依存関係・リスク)
- (d) 未確定事項・オーケストレーターへの確認事項

なお、これは Codex 連携スクリプト(T081)の動作確認用のデモ依頼です。
実際にタスク化する前提の調査ではなく、スクリプトが正しくレポートを生成できるかの検証が目的です。
